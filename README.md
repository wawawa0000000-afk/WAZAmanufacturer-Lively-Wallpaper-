# -輪座-製作所 Wallpaper (Waza Manufacturing Wallpaper)

-輪座-製作所（Waza Manufacturing）が提供する、Lively Wallpaper向けのカスタム動く壁紙（GLSL Shader Animation）です。
ミニマルなタイポグラフィと、動きに合わせてRGB/CMYの色乖離（クロマ・ディレイ）が発生するサイバーなエフェクトを特徴としています。

## Features (特徴)
* **GLSL Shader Animation**: 軌跡を描く円と、残像による色ズレエフェクト。
* **Retina / High-DPI Support**: デバイスのピクセル比率（DPR）を自動取得し、常にシャープで美しい描画を実現。
* **Theme Switching**: `index.html`のコメントアウトを1箇所変更するだけで、白テーマ（シアン・マゼンタ・イエローの残像）と黒テーマ（赤・緑・青の残像）を切り替え可能。

## Installation (導入方法)
1. このリポジトリをダウンロード（ZIP）、またはCloneします。
2. Windows PCに [Lively Wallpaper](https://rocksdanister.github.io/lively/) をインストールします。
3. Lively Wallpaperを起動し、ダウンロードしたフォルダの中にある `index.html` をウィンドウ内にドラッグ＆ドロップするだけで適用されます。

## Customization (テーマの変更)
デフォルトでは黒テーマが適用されています。白テーマに変更したい場合は、`index.html` の末尾にある以下の部分を書き換えてください。
```html
<!-- 白テーマにする場合 -->
<script src="waza.js"></script>
